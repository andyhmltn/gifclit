GIF.CL.IT                    General Commands Manual                GIF.CL.IT(1)

NAME
     gif.cl.it -- retrieve a gif image for a search query

SYNOPSIS
     http://[query].gif.cl.it/

DESCRIPTION
     The gif.cl.it service takes [query] and asks 3rd party services to find an
     appropiate animated GIF image for said query. No filters or restrictions
     are made on the explicitness of returned images.

EXAMPLES
     The request:

            http://winning.gif.cl.it

            or

            http://gif.cl.it/winning 

     viewed in a browser will likely return an image of Charlie Sheen.
